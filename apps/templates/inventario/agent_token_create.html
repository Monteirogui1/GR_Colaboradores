Como funciona

🔑
Um token, várias máquinas

O mesmo token pode ser instalado em múltiplas máquinas. Cada máquina é registrada individualmente no sistema.

🔒
Seguro e rastreável

O token não é salvo em disco — só o hash SHA-256. Cada uso é registrado com hostname e data.

⏱
Validade configurável

Tokens expirados não podem registrar novas máquinas. Máquinas já registradas continuam funcionando normalmente.

👁
Visível apenas uma vez

O token em texto puro é exibido só na geração. Guarde-o ou distribua imediatamente para as máquinas.
